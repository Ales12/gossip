# Gerüchteküche

Bei diesen Plugin können User Gerüchte über andere in die Welt streuen. Hierbei kann ausgewählt werden, ob es eine ganze Gruppe betrifft oder nur gewisse Charaktere.

## Link
misc.php?action=gossip
